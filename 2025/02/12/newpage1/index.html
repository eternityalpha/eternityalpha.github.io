尝试使用hexo书写博客（不懂问deepseek）

hexo命令相关

  • 在blog根目录下打开CMD窗口，输入hexo new newpage1（hexo new page np1,创建新页面）,找到/source文件下同名的markdown文件，进行编写。
  • 编写完成后，使用命令更新本地hexo网页。
  • 输入命令 hexo clean hexo generate hexo server 检测本地网页是否修改成功。
  • 输入命令 hexo 命令 --debug 对命令进行详细步骤输出 （报错时一定要使用该命令定位问题文件）
